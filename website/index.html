動作環境 Requirements

  • Java 8
  • Spigot 1.9
  • ProtocolLib 3.7.0-BETA

上記の環境に合わせて作られていますが、それ以外の環境でも問題無く動く場合があります。
CraftBukkit 1.8.8などでも問題無く動作するはずです。

コマンド Commands

/heads 全てのヘッドを表示します
/heads [辞書名] 辞書名に対応したヘッドを表示します
/heads [検索クエリ] キーワードを元に検索します
正規表現を利用出来ます

辞書 Dictionary

プラグインをインストールすると、plugins/HeadDictionary/headsと言うディレクトリが作成されているはずです。
そのディレクトリの中にある.jsonで終わるファイルが辞書データとして読み込まれます。

デフォルト辞書 Default Dictionary

プラグインをインストール直後のplugins/HeadDictionary/headsには、12個のjsonファイルが入っているはずです。 それがデフォルト辞書です。
入っていない場合は、ここから手動で入手することが可能です。
デフォルト辞書は、Quality Headsから HeadCollectionsFetcherを使って抽出されたもので、 現在12個の辞書があり、合計で800個を超えるヘッドが記録されています。

辞書のカスタマイズ Customize dictionary

辞書ファイルは単純なjsonファイルなので、簡単にカスタマイズすることが可能です。
デフォルト辞書を参考にして自由にカスタマイズしてください。
